Welcome to Fhish! This guide will help you set up your first private FHE rollup on the Initia ecosystem using our interactive CLI.

## Prerequisites

- **Docker Desktop**: Required for service orchestration.
- **Go 1.22+**: Only if you intend to build the CLI from source.
- **Git**: To clone configuration files.

## Installation (v0.1.8)

Download the latest Fhish CLI binary for your platform:

```bash
# macOS (Silicon)
curl -L -o fhish https://github.com/fhish-tech/fhish-cli/releases/download/v0.1.8/fhish-darwin-arm64
chmod +x fhish
sudo mv fhish /usr/local/bin/
```

## Launching your Rollup

The `fhish create all` command is the entry point for your private environment. It launches an interactive wizard that guides you through:

1.  **Chain Setup**: Choosing your Rollup name and deriving a deterministic EVM Chain ID.
2.  **Infrastructure**: Provisioning a MiniEVM node.
3.  **Privacy Layer**: Setting up the FHE Gateway and generating cryptographic evaluation keys.
4.  **Relay Services**: Configuring the Relayer for encrypted state transitions.

```bash
fhish create all
```

## Verifying the Protocol

After the stack is running, you can perform an end-to-end verification to ensure the FHE co-processor is correctly decrypting transactions:

```bash
fhish docker verify
```
