The Fhish architecture consists of several key components working together to enable private computation on Initia.

## Key Components

1. **Fhish MiniEVM Node**: A specialized Initia rollup node with FHE precompiles.
2. **FHE Gateway**: An off-chain service that manages FHE keys and performs final decryptions.
3. **FHE Relayer**: Bridges the gap between the on-chain FHE requests and the off-chain Gateway.
