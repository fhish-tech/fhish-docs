---
title: The FHE Relayer
description: Automating the bridge between rollup and gateway.
---

The FHE Relayer automates the flow of data in the Fhish stack. 

- **Listening**: Monitors the MiniEVM for FHE computation requests.
- **Forwarding**: Sends encrypted data to the Gateway for processing.
- **Callback**: Returns the results to the smart contract on-chain.
